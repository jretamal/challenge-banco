Utilizamos SDD definiento un documentos docs/instrucciones.md dentro de cada repositorio para especificar los requerimientos desde stack, objetivo hasta documentacion.

Se utilizo Antigrvity CLI (agy) usando el modelo Gemini 3.5 Flash (Medium)

Se crearon 3 proyectos:

Angular 19:
Repositorio: https://github.com/jretamal/challenge-angular

Express:
Repositorio: https://github.com/jretamal/challenge-express

NestJS:
Repositorio: https://github.com/jretamal/challenge-nestjs


Para el uso de Skills se uso:

https://www.skills.sh

Para angular usamos la skills:
npx skills add https://github.com/mindrally/skills --skill angular

Para express la skills:
npx skills add https://github.com/wshobson/agents --skill nodejs-backend-patterns

Para NestJS la skills:
npx skills add https://github.com/kadajett/agent-nestjs-skills --skill nestjs-best-practices

Cada skill quedo instalada en cada carpeta del proyecto.
